Решение квадратных уравнений

Введите коэффициенты a, b и c для уравнения: ax² + bx + c = 0

Пример: 2x² - 5x + 3 = 0 → a=2, b=-5, c=3

Решение:
Введите коэффициенты

Как это работает:

Дискриминант вычисляется по формуле: D = b² - 4ac

  • Если D > 0: два корня: x₁,₂ = (-b ± √D) / (2a)
  • Если D = 0: один корень: x = -b / (2a)
  • Если D < 0: нет действительных корней

Решение линейных уравнений

Решает уравнения вида: Ax + B = C или A + Bx = C

Укажите, какие числа содержат переменную x

Ваше уравнение:

Введите числа и отметьте, какие содержат x
Решение:
Введите данные уравнения

Примеры:

Пример 1: 3x - 1 = 0
  • c₁ = 3 (содержит x ✓)
  • c₂ = -1 (не содержит x)
  • c₃ = 0
  • Ответ: x = 0.333...
Пример 2: 5 + 2x = 11
  • c₁ = 5 (не содержит x)
  • c₂ = 2 (содержит x ✓)
  • c₃ = 11
  • Ответ: x = 3
