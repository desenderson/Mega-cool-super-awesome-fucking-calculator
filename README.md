# Mathematical calculators

A web application with three types of mathematical calculators:
1. **Ordinary calculator** - calculation of mathematical expressions
2. **Quadratic Equation Calculator** - solution ax2 + bx + c = 0
3. **Linear equation calculator** - solving equations of the form Ax + B = C

## Functionality
- Calculation of complex expressions with support for parentheses
- Automatic solution of quadratic equations with discriminant output
- Solving linear equations with flexible input
- History of all calculations
- Responsive design for mobile devices

## Technology
- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
- Git for version control
